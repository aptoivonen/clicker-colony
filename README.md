## About The Project

Clicker Colony is a clicker-base-building-game. You are the leader of a mining colony on a distant planet, tasked with the mining of minerals and processing them into high-quality materials. Hazardous environments, aliens, and competitors pose challenges.

## Technologies

Bootstrap, javascript, parcel, postcss, purgecss, autoprefixer
